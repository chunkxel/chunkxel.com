Hey there! Welcome to the Chunkxel Network's

Chunkxel.com is a bleeding-edge beta Minecraft network. We're currently running
Minecraft version 1.14.4 with PaperMC and Waterfall Proxy
We make use of a high-performance docker-based management daemon
and use WireGuard as an interconnect layer.

We currently only operate a single game-mode, township, but with donations plan to expand.
